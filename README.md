SelectBox-mobile
======

jQuery plugin: select box for mobile use, custom ui and use native os option menu.

###HTML

```
<select>
	<option>option 1</option>
	<option>option 2</option>
	<option>option 3</option>
</select>
```

###jQuery

```
$('select').selectboxMobile();
```
